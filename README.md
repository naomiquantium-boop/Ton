# SpyTON BuyBot (TON)

Clean TON build for SpyTON.

Included:
- TON payment verification with memo/comment matching
- TON jetton buy tracking
- buy posting for groups and channel
- STON.fi / DeDust / custom buy-link template support
- trending invoices with memo verification
- ads invoices with memo verification
- owner commands kept intact
- trending leaderboard updater

Trending prices:
- TOP3: 2h 7 TON, 4h 14 TON, 8h 21 TON, 24h 35 TON
- TOP10: 2h 5 TON, 4h 10 TON, 8h 17 TON, 24h 30 TON

Ads prices:
- 1 day 10 TON
- 3 days 25 TON
- 7 days 60 TON

Core TON helpers now live in `utils/ton_rpc.py`.
